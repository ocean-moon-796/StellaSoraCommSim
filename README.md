ステラソラ依頼シミュレーター
YostarのRPGゲーム「ステラソラ」内の「依頼」における最適な組み合わせを自動で出力するシミュレーターツールです。
派遣する依頼をアイテムアイコンから任意の数選択し、下部で未所持、または基準レベルに満たない巡遊者を選択し検索から除外して検索を行います。
